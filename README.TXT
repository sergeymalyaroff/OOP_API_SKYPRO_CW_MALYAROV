инструмент на Python, который позволяет пользователям поиска вакансий на двух платформах:
HeadHunter (hh.ru) и SuperJob (superjob.ru).
Это приложение использует API каждого сайта для поиска вакансий по определенному запросу,
после чего результаты сохраняются в файл JSON для последующего использования.
